# AI Facial Recognition Attendance System

## Description
A smart, automated attendance system that uses Face Recognition, OpenCV, and Deep Learning to identify employees from live camera feed and mark their attendance in real time. Designed for industries to ensure fast, secure, and accurate attendance tracking without manual effort.

## Features
- Add new employees with image dataset  
- Capture and store live face images  
- Detect and encode multiple faces using Deep Learning  
- Maintain the total user count in database  
- Single-click “Take Attendance”  
- Recognizes multiple faces at once  
- Auto-marks attendance for all detected users  
- Displays live attendance reports
